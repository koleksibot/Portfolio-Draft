Hero Mini Game

OVERVIEW
This game is similar to the once popular game "flappy bird". I wanted to try creating a 2D game using the HTML canvas element, and practice animating/clipping spritesheets. I created the flying hero spritesheet using piskel. On desktop, the game allows the hero to move back, forward, up, and down. This is different from Flappy Bird and adds a layer of complexity to the game.

Meridian Rehabilitation

OVERVIEW
Meridian Rehabilitation is an occupational therapy firm with several locations in the Lower Mainland. Meridian wanted a design upgrade to attract new staff members and streamline the process of making referrals for clients. I designed the logo and helped style the website using css on Wordpress. The website has a simple UI, and it is much easier to navigate and find information compared to the old site.
